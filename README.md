# opencage-api-client-nextjs-test

## Prerequisites

- node LTS
- pnpm

## Getting Started

First, clone the repo, install the dependencies run the development server:

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
